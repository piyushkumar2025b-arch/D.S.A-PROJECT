# 🔬 3. Section 2: Live Research Agent

**Section 2** introduces the concept of a hierarchical AI system designed to resolve complex academic and logical problems.

### ⚙️ How to Use It
1. Once selected, you'll see the **5-Agent Academic Setup**.
2. **Input Layer**: Type in a deeply complex thesis inquiry.
3. **Execution**:
   - The system utilizes the **Master Synthesizer** pattern.
   - It will boot up 5 unique system prompts acting as 5 unique researchers.
   - Each outputs their dedicated perspective.
   - The system gathers all 5 outputs and feeds them back into a final Master Agent to write the unified Executive Report.

### 🎯 Primary Use Cases
* Academic paper summarization.
* Deeply analytical structural breakdowns where relying on one LLM hallucination is dangerous. Multiple agents debating the prompt yields exponentially higher logical accuracy.
